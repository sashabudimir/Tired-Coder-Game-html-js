Caffeine Quest: Survive the Workday 


■ Game Story
You are the Exhausted Dev, a tired programmer trapped in the gloomy open office of Scriptoria Inc..
Rumor has it that deep inside the Server Room lies the Sacred Espresso Shot, guarded by the fearsome Espresso Demon.

Your mission is simple but dangerous:
Survive the workday, defeat your caffeine-based enemies, and reclaim your focus before your motivation runs out.


■ Gameplay Mechanics 
This is a text-based adventure built entirely in JavaScript using Objects, Classes, and the Lodash library.

Core Classes:
- Character (Player and NPCs)
- Item (Weapons, Healing, Quest items)
- Location (Game areas and connections)

Core Features:
- Explore multiple rooms (Open Office → Break Room → Server Room)
- Pick up and use items
- Fight enemies and talk to NPCs
- Two win conditions: defeat the demon or find the sacred espresso

Hidden commands: coffee (boost) and stat (debug info).

Controls / Commands 
- look – Re-describe your location
- search – Search for items
- pickup [item name] – Collect items
- inventory – View carried items
- use [item name] – Use healing or weapon
- attack [character name] – Fight NPCs
- talk [character name] – Speak to NPCs
- move [location name] – Move to another area
- help – View commands
- quit – End the game


■ How to Run 
1. Open index.html in your browser.
2. The intro appears automatically.
3. Type commands in the box and press Enter or click Run.
4. Play until you win. 


■ Technologies 
- HTML, CSS, JavaScript
- Lodash library
- Object-Oriented Programming
- DOM Manipulation
